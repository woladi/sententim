# Changesets

Drop one Markdown file here per change. Use `pnpm changeset` (or write by hand) — the format is the [standard changeset](https://github.com/changesets/changesets/blob/main/docs/intro-to-using-changesets.md).

```md
---
"sententim": minor
---

Added CJEU ingestion via CELLAR SPARQL.
```
